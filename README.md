# ATM AI Security System

An AI-based ATM surveillance system to detect suspicious activities such as:

- Face concealment (helmet / mask)
- Multiple people near ATM
- Camera tampering (blackout)

## Features

- Real-time camera monitoring
- Modular detection & alert system
- Event-based alert triggering
- Designed for edge deployment

## Tech Stack

- Python
- OpenCV
- YOLO (Ultralytics)
- Modular ML architecture
